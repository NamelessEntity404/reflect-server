Always end every completed task with a link to check the result in the browser.

**Why:** User expects a link at the end of every task so they can immediately verify the work without having to ask.

**How to apply:** After any code change that affects the browser, end the response with the relevant localhost URL. For this project, the site runs at http://localhost:8765.
